A riddle found in ancient treatise, written circa 3rd-5th century, as inscribed below:
I think a number, unknown to you.

If you count by threes, there is a remainder 2.
If you count by fives, there is a remainder 3.
If you count by sevens, there is remainder 2.

Could you tell what my number is ?

Input data: a list of tuples, whose tuple contains remainder and its divisor

Output data: The output will be a number, the smallest possible natural number.
